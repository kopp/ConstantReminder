# Constant Reminder

![Constant Reminder Logo](ConstantReminderLogo.png)

This Android App helps you to remember things by constantly reminding you of them.
It reminds via notifications which come more or less often depending on what the user decies.

All state of the app is exportable to and importable from JSON, so you can easily share Reminders.

See [releases](https://github.com/kopp/ConstantReminder/releases) for up-to-date APKs.
