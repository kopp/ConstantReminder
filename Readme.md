# Constant Reminder

![Constant Reminder Logo](ConstantReminderLogo.png)

This Android App helps you to remember things by constantly reminding you of them.
It reminds via notifications which come more or less often depending on what the user decies.

All state of the app is exportable to and importable from JSON, so you can easily share Reminders.
Use this feature to backup your configuration and to format longer text on a PC/Laptop.

See [releases](https://github.com/kopp/ConstantReminder/releases) for up-to-date APKs.

# Constant Reminder (Deutsch)

Diese Android-App hilft dir dabei, dich an Dinge zu erinnern, indem sie dich ständig daran erinnert.
Die Erinnerung erfolgt über Benachrichtigungen, die je nach Entscheidung des Nutzers öfter oder seltener erscheinen.

Der gesamte Zustand der App kann als JSON exportiert und importiert werden, sodass du Erinnerungen einfach teilen kannst.
Nutze diese Funktion, um deine Konfiguration zu sichern und längere Texte bequem an PC oder Laptop zu formatieren.

Aktuelle APKs findest du unter [Releases](https://github.com/kopp/ConstantReminder/releases).
